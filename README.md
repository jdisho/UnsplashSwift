# UnsplashSwift
🌯 A Swift wrapper for the Unsplash API

## 🛠 Installation

### CocoaPods

To integrate UnsplashSwift into your Xcode project using CocoaPods, specify it in your `Podfile`:

```ruby
pod 'UnsplashSwift', '~> 0.1'
```
